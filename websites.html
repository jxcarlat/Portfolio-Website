Capstone Project

I worked with one other team member to develop a Website featuring PHP and Unity game engine interestingly enough. Unity was used for the client-side and functioned quite well for the most part, however you can tell quickly that it was meant for running games online and not for client-side web programming. This is a scheduler website that assigns teachers stored in the database to teach certain classes based on skills and time slot availability. I was mainly responsible for the PHP and creation of the database while my teammate was responsible for the algorithm as well as the Unity code. Also the CSS is a little lacking but not bad for making a full functional website for the first time.
